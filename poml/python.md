POML Python SDK[¶](https://microsoft.github.io/poml/stable/python/#poml-python-sdk "Permanent link")
====================================================================================================

_Estimated time to read: 1 minute_

This documentation provides an overview of the POML Python SDK, which allows you to work with POML files and components in your Python projects.

Installation[¶](https://microsoft.github.io/poml/stable/python/#installation "Permanent link")
----------------------------------------------------------------------------------------------

### Stable Release[¶](https://microsoft.github.io/poml/stable/python/#stable-release "Permanent link")

```
pip install --upgrade poml
```

### Nightly Build[¶](https://microsoft.github.io/poml/stable/python/#nightly-build "Permanent link")

```
pip install --upgrade --pre --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ poml
```
