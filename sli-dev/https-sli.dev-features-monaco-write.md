Writable Monaco Editor [â€‹](http://sli.dev/features/monaco-write#writable-monaco-editor)
=======================================================================================

You can also use the [Import Code Snippets](http://sli.dev/features/monaco-write#import-code-snippets) syntax combined with the `{monaco-write}` directive, to link your Monaco Editor with a file on your filesystem. This will allow you to edit the code directly in the editor and save the changes back to the file.

md`<<< ./some-file.ts {monaco-write}`

When using this, be sure to back up your files beforehand, as the changes will be saved directly to the file.
